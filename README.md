## Tools

### Kraftfeldanalyse nach Kurt Lewin

**[→ Tool öffnen](https://ewingo13.github.io/lern-tools/kraftfeldanalyse.html)**

Eine interaktive Kraftfeldanalyse: Treibende und hemmende Kräfte gegenüberstellen, gewichten und live visualisieren.

Der didaktische Kern: Veränderung gelingt leichter durch weniger Widerstand als durch mehr Druck. Mehr Druck erzeugt mehr Gegendruck. Der eigentliche Hebel ist, hemmende Kräfte abzubauen.

**Funktionen:**
- Treibende und hemmende Kräfte als skalierbare Pfeile
- Schieberegler 0 bis 10 je Kraft
- Bezeichnungen editierbar, Kräfte hinzufügen und entfernen
- Animierte Balkenwaage als visueller Indikator
- Netto-Ergebnis mit didaktischem Hinweistext
- Läuft vollständig im Browser, keine Daten werden erhoben
